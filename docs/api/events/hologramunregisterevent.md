title: HologramUnregisterEvent
description: 'Extends DecentHologramsEvent'

/// info | New since 2.9.6
///

This event is called whenever a Hologram is unregistered through `HologramManager#unregister(Hologram)`.

## Methods

| Method          | Description                              |
|-----------------|------------------------------------------|
| `getHologram()` | Gets the Hologram that got unregistered. |
