---
title: '/dh displays move'
description: 'Moves the Display Entity'
---

The `move` subcommand moves the provided Display Entity to the specified coordinates.

{{ permissions("displays", "move") }}

{{ aliases(["mv"]) }}

## Arguments

### `#!command <name>`

Name of the Display entity to move.

### `#!command <x>`

The X coordinate to move the Entity to.

### `#!command <y>`

The Y coordinate to move the Entity to.

### `#!command <z>`

The Z coordinate to move the Entity to.
