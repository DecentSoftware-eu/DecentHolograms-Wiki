---
title: '/dh displays list'
description: 'Lists all currently created Display Entities'
---

/// warning | This command is only usable by Players!
///

The `list` subcommand lists all currently existing Display Entities.  
Each entry contains the Display's name and location.

{{ permissions("displays", "list") }}

## Arguments

### `#!command [page]`

Page number in the list to show.
