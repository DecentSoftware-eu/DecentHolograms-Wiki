---
title: '/dh hologram move'
description: 'Teleports a Hologram to specific coordinates'
---

The `move` subcommand is used to teleport a Hologram to a set of coordinates.

{{ permissions("hologram", "move") }}

{{ aliases(["mv"]) }}

## Arguments

### `#!command <hologram>`

Name of the Hologram to teleport to.

### `#!command <x>`

The X coordinate to teleport the Hologram to.

### `#!command <y>`

The Y coordinate to teleport the Hologram to.

### `#!command <z>`

The Z coordinate to theleport the Hologram to.
