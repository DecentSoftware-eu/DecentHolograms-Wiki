---
title: '/dh hologram lines'
description: 'Shows lines of a Hologram'
---

The `lines` subcommand is used to show all the lines of a specific hologram page.

{{ permissions("hologram", "lines") }}

## Arguments

### `#!command <hologram>`

Name of the Hologram to list lines of.

### `#!command <page>`

The Hologram page to list lines of.

### `#!command [listPage]`

Optional page number for the list itself.  
This is only useful when there are more lines than what is displayed on one page in chat.
