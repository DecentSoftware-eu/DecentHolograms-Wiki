---

The `removeflag` subcommand is used to remove a Flag from a Hologram.  
For a list of Flags, check the Flags page.

{{ permissions("hologram", "removeflag") }}

## Arguments

### `#!command <hologram>`

Name of the Hologram to remove a flag from.

### `#!command <flag>`

Name of the flag to remove.
