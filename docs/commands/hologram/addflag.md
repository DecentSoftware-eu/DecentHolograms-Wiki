The `addflag` subcommand allows to add a Flag to a Hologram.  
See the Flags page for available Flags.

{{ permissions("hologram", "addflag") }}

## Arguments

### `#!command <hologram>`

Name of the Hologram to add a Flag to.

### `#!command <flag>`

Name of the Flag to add.
