---
title: '/dh lines addflag'
description: 'Adds a flag to a Hologram line'
---

The `addflag` subcommand adds a flag to a Hologram line.

{{ permissions("lines", "addflag") }}

## Arguments

### `#!command <hologram>`

Name of the Hologram to add a flag to.

### `#!command <page>`

Page in the Hologram to add a flag to.

### `#!command <line>`

Line number in the Hologram to add a flag to.

### `#!command <flag>`

Name of the Flag to add to the Line.
