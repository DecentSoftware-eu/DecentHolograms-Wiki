---
title: '/dh lines edit'
description: 'Retrieve a pre-made command to edit a line'
---

/// warning
This command is only usable by players!
///

The `edit` subcommand allows you to get a Clickable text, providing you with a pre-filled `/dh lines set ...` command to edit the defined Line more easily.

{{ permissions("lines", "edit") }}

{{ aliases(["e"]) }}

## Arguments

### `#!command <hologram>`

Name of the Hologram to edit a line of.

### `#!command <page>`

Page in the Hologram to edit a line of.

### `#!command <line>`

Line number to retrieve an edit command for.
